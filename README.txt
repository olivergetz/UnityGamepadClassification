=== AI DUALSENSE MUSIC CONTROLLER ===

Author: Oliver Getz
Date: 02/04/2023

=== DESCRIPTION ===

The purpose of this project is to demonstrate one way AI can be used to create interactive music systems for video games.

=== REQUIREMENTS ===
- Unity 2020.3.28f or later
- Barracuda 3.0 for Unity
- Rewire input controls for Unity (needed for accelerometer data)
- A gamepad controller with accelerometer and gyroscope (Dualsense for PS5 recommended)

=== HOW TO USE ===

1. Connect your gamepad controller
2. Hit play in Unity
3. Move the controller around
4. Inspect the console to see predictions and listen to the music change
